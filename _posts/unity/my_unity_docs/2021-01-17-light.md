# 👩‍🦰 Light

> 이 컴포넌트가 붙으면 해당 오브젝트는 빛을 낼 수 있다. 오브젝트에 붙어 빛을 발산

파티클 입자에 붙여 빛나는 이펙트 효과를 줄 수도 있다. 전등 같은 것도 구현할 수 있을 것 같다.

## 🚀 에디터

- Range : 빛이 도달할 사정 거리
- Color : 빛의 색
- Intensity : 빛의 강도
