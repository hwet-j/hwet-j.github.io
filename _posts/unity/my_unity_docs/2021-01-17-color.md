## 🚀 함수

### ✈ Lerp

> public static Color Lerp(Color a, Color b, float t);

- a, b 색깔 사이의 t (0~1) 비율에 위치한 색을 리턴한다.
  - t 가 0.5라면 리턴되는 값은 a 와 b 의 중간값!
