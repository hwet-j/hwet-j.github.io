## 잡다 기능

### 대,소문자로 변경하기

문자열의 경우에는 toUpperCase(), toLowerCase()를 사용하여 쉽게 변경가능하다.

> 사용 예시
```java
String str = "asndFaDAna";

System.out.println(str.toLowerCase()); // -> asndfadana
System.out.println(str.toUpperCase()); // -> ASNDFADANA
```

### 특수문자 출력하기

자바에서 사용되는 이스케이프 문자는 / 이다.
